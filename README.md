# IPZ-MANIFEST

## Init project:
```bash
  repo init -u https://github.com/ipz-automotive/ipz-manifest -b main -m default.xml
```

## Sync project:
```bash
  repo sync -j8
```

## Manifest includes:
- googlesource/ - AOSP branch android-14.0.0_r20
- raspberry-vanilla/ - AOSP for RPI4 branch android-14.0
- ipz-automotive/ - project repos
